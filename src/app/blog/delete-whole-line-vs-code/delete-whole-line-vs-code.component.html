The following was tested using VSCode version 1.64.1.

Have you ever needed to delete some code? No? Why not? Try it out... it's therapeutic!

Once you are in need to delete some code, you have two scenarios:

  1. Multi line deletion.
  2. Single line deletion.

Scenario #1: Easy peasy, right? go to the end of the last line you want to delete, hold shift key, move the cursor to the beginning of the line that wraps all you want to delete and hit delete key. Done and done!

Now, if you are doing the same thing for deleting single lines, let me tell you a little secret: there's an easier and faster way.

Just, place the cursor in the line you want to nuke; make sure you are not selecting anything... I mean your cursor should be just blinking normally, and then press the keyboard shortcut for cut and voila! line will disappear.

  • Mac OS: cmd + x
  • Linux & Windows: ctrl + x

Note: If you are selecting something, the outcome should be no surprise to you... you will cut whatever you are selecting which is fine, but not what we intended to do.
