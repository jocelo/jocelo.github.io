Bienvenido developer!

En este corto articulo, me gustaría compartirte un tip de productividad relacionado con VS Code: Como abrir VSCode desde una terminal y automáticamente abrir también cualquier directorio en el que te encuentres.

Breve y consiso: abre una terminal y executa el siguiente comando:

> code .

Básicamente code abre VS Code y con el único parámetro que le agregamos: . le estamos indicando que por favor incluye el directorio actual cuando abras VS Code.

No funcionó? Eso significa que no tienes instalado el comando, pero no te preocupes, ahora veremos como se instala:

  • Con VSCode abierto, abre la Paleta de Comandos. La encuentras en el menu View (o presiona shift+command+p en mac).
  • Comienza a escribir: install code command in path y presiona la tecla enter.
  • Recibirás un mensaje de confirmación cuando el comando haya terminado de instalarse.

Fácil, verdad? Personalmente uso este comando todo el tiempo, espero encuentres util este tip. Nos leemos pronto!
