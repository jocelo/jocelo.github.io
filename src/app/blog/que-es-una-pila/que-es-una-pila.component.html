Definición

La Pila (o stack) es una estructura de datos o tipo de datos abstracto que implementa el principio LIFO.

LIFO significa Last Input First Output; en otras palabras es el ultimo elemento que entra, es el primero que sale.

Una analogía que sirve para comprender mejor este concepto seria sacar una papa de un tubo de Pringles... cuando queremos comernos una papa, la primera papa que tomamos es la ultima papa que metieron al contenedor cuando ensamblaron el producto.

Otro ejemplo es el historial de un navegador web (Firefox, Chrome, etc) en donde cada vez que das click a un link, te lleva a esa pagina, y te pone en el historial la pagina que estas abandonando. De esta manera, cuando presionas el botos de "back", te regresa a la ultima pagina que habías visitado, y si continuas dando click en el botón, vas navegando "hacia atrás"

O bien, el ejemplo clásico... una pila de platos, los cuales solo puedes tomar el plato que esta mas arriba de la pila (siempre y cuando no quieras tomar también la escoba y el recogedor).

Asi pues, las operaciones básicas que se pueden ejecutar en una pila son la inserción de un nuevo elemento y la obtención de un elemento. Estas dos operaciones generalmente se les llama push y pop respectivamente.

Para su implementación todo depende del lenguaje de programación y es probable que todos los lenguajes tenga este tipo de dato previamente programado y solo tengamos que aprender a utilizarlo; dichos métodos son definidos como parte de un array (o un Lista en Python).

Las operaciones de una pila siempre son efectuadas sobre el tope de la pila ó la cabeza de la pila. Termino que se utiliza para referirse a el final de la lista.

Análisis Complejidad (big O)

La pila es un tipo de dato abstracto muy usado en programación. El tiempo de lectura y escritura en cuestión de complejidad es constante ya que todas las operaciones (inserción y obtención) son realizadas sobre el tope de la pila; esto hace que la complejidad sea constante: O(1).

Conclusión

Gracias por leer este articulo!

La pila es un tipo de dato abstracto que es muy recurrida cuando estamos resolviendo problemas de optimización. Generalmente es muy sencilla de implementar y el tipo de dato array (o una lista en Python) en los lenguajes de programación usualmente ya cuentan con métodos para usar el array como si fuera una pila.

Este es el tipo de dato perfecto para resolver problemas de programación como el del balanceo de paréntesis.

Listo! ya conoces acerca de este tipo de dato abstracto que puedes usar cuando estés programando.
