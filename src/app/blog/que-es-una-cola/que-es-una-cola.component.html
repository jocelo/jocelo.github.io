Definición

La Cola (o queue) es una estructura de datos o tipo de datos abstracto que implementa el principio FIFO.

FIFO significa First Input First Output; en otras palabras el primer elemento que entra es el primero que sale.

Una analogía que sirve para comprender mejor este concepto es como funciona una fila de personas en el cine, o en las tortillas... La primera persona que llego es la primera en ser atendida cuando comienzan a pasar al cine o a vender las tortillas.

Las operaciones básicas que se pueden ejecutar en una pila son la inserción de un nuevo elemento y la obtención de un elemento. Estas dos operaciones generalmente se les llama push y pop respectivamente.

Para su implementación todo depende del lenguaje de programación y gran parte de los lenguajes tenga este tipo de dato previamente programado y solo tengamos que aprender a utilizarlo; dichos métodos son definidos como parte de un array (o un Lista en Python).

Las operaciones de una cola siempre son efectuadas sobre el inicio de la cola ó la cabeza de la cola. Termino que se utiliza para referirse a el inicio de la lista.

Análisis Complejidad (big O)

La cola es un tipo de dato abstracto muy usado en programación. En su implementación básica tiene una complejidad para lectura y escritura de O(n); donde n es el tamaño de la cola. Esta complejidad esta basada en que cada vez que se saque o inserte un elemento de la cola, la operación se realiza sobre el tope de la cola y todos los elementos restantes tienen que recorrerse hacia la izquierda o derecha, respectivamente.

Nota: El termino lectura en el párrafo anterior se refiere a la operación pop. El termino no debe confundirse con algún otro método u operación de lectura ( peek por ejemplo ). Por cierto, dicho método ( seek ) tiene una complejidad constante O(1), ya que solamente se esta leyendo el elemento en el tope de la cola, lo cual no requiere recorrer ningún otro elemento de la misma. Este método (o alguna variante) para trabajar con colas existe en algunas implementaciones o clases incluidas en algunos lenguajes de programación. Usando estas clases, ayudan a reducir la complejidad hasta que sea constante O(1). Por ejemplo, python (python2.7 y superiores) cuentan con un objeto llamado deque (double-ended queue) que es parte del modulo collections; el cual ofrece aproximadamente el mismo comportamiento de complejidad constante O(1) en operaciones de inserción y obtención. Puedes leer mas acerca de este objecto en la documentacion oficial.

Conclusión

Gracias por leer este articulo!

La cola (en conjunto con la pila ) es un tipo de dato abstracto muy recurrida para resolver problemas o algoritmos de optimización. Generalmente es muy sencilla de implementar y el tipo de dato array (o una lista en Python) en los lenguajes de programación usualmente ya cuentan con métodos para usar el array como si fuera una cola. O bien, como vimos anteriormente, utilizando módulos creados específicamente para este caso; inclusive podríamos crear una clase nosotros mismos para mejorar la funcionalidad de una cola.

Listo! ya conoces acerca de este tipo de dato abstracto que puedes usar cuando estés resolviendo problemas.
