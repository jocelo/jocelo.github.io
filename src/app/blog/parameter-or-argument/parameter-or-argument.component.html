The terms Parameter and Argument are interchangeable, right? Both mean the same thing: The stuff/value/variables that is being passed into a function.

If you search the web, you might found some articles that mention both terms can be used interchangeably. And that is the way that I have being using them pretty much since I began writing code (well, I began writing HTML but you know what I mean)... What if I told you that there is a subtle but important difference?.

via GIPHY

Ok, so first, let us define each terms: according with Wordnik and Wikipedia :

Parameter: A variable used in a calculation within a computer program which must be assigned a value before the calculation can be performed.

Argument: A value, or reference to a value, passed to a function.

So, with this new knowledge we can say:

If you think about it (and use your muscle memory) at some point in the past you probably would have said something like: if these optional parameters do not have default arguments we are creating a bug.

Aha! there you have it! You might have already being using the correct terms but never thought about it outside this context.
